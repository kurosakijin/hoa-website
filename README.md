# HOA Site

Homeowners association management starter built with the MERN stack, React, Tailwind, and SCSS.

## Apps

- `client/`: React + Vite frontend with dashboard, residents, payments, and admin views.
- `server/`: Express API with MongoDB-ready models and mock data fallback.

## Run locally

```bash
npm install
npm run dev
```

## Environment

Create these files before running against a real backend:

- `client/.env` from `client/.env.example`
- `server/.env` from `server/.env.example`
